Write everything in English — code, comments, log messages, hook output, memory entries.

Only add comments for non-obvious design decisions. Never comment self-explanatory code.

**Why:** Consistency across the codebase and team. Clean code should be readable without comments.

**How to apply:** All new files, hook scripts, and Claude responses use English. Strip unnecessary comments. Let naming and structure convey intent.
