**Critical — why a plain pandoc conversion is wrong here.** `pandoc --reference-doc=<template>` copies ONLY the reference document's *styles* (plus headers/footers and section properties); it DISCARDS the reference document's *body*. A straight `pandoc … --reference-doc=<template> -o out.docx` therefore throws away the template's cover page and TOC page — the "content gets deleted" symptom. So DO NOT output the final DOCX directly from pandoc. Convert the markdown to a **body-only** DOCX (template as reference, for styles), then **merge** that body into a fresh **copy of the template** (which still holds the cover page, the Word TOC field, headers/footers). Merge, never replace.

Rules:
- **Heading numbering (avoid double numbering)**: `ProposalResult.md` carries manual chapter numbers in its heading TEXT (`## 1 Ausgangssituation`, `### 2.1 …`). The template's heading styles (`Überschrift1`–`9`) ALSO apply automatic Word list-numbering (`numPr`); if both are active the DOCX shows doubled numbers ("1 1 Ausgangssituation"). The manual markdown numbers are authoritative, so after the merge DISABLE the template's automatic heading numbering: for every heading style in the merged document remove its `numPr`, and clear any direct-paragraph `numPr` on heading paragraphs. Verify no heading shows a duplicated number.
- **Native tables**: read the markdown with a reader that parses pipe tables (`from='gfm'` or `markdown+pipe_tables`) so they become real Word tables. Never rasterize or flatten tables to text.
- **Table of contents**: keep the template's native Word TOC field (it restyles and updates on open). Do NOT pass pandoc `--toc` (that injects a second, static list). If the chosen template has NO TOC field, insert one (`TOC \o "1-3" \h \z \u`) right after the cover page before merging.
- **Readability — page breaks**: after the merge, insert a page break before every top-level chapter (`Überschrift1`/`Heading 1`) except the first, so chapters start on a fresh page. Rely on the template's paragraph-spacing styles for intra-section whitespace; do not hand-tune margins.
- **Cover-page addressee**: after the merge, set the client/recipient placeholder on the (now preserved) template cover page to `client_name`, verbatim from `ClientContextResult.json` (full organisation name including legal form — NOT the filename slug), keeping exact spelling/casing. If `client_name` is missing, empty, or `"Unknown_Client"`, leave the template default placeholder untouched.

```python
import re, pypandoc
from docx import Document
from docx.oxml.ns import qn
from docx.oxml import OxmlElement

TEMPLATE = "/opt/assets/templates/docx/service_proposals/<selected_template>.docx"
OUT = "Proposal_<slug>.docx"

# 1. Encoding hygiene safety net (orchestrator already ran it): assert no 0x08
with open("ProposalResult.md", encoding="utf-8") as f:
    md = f.read()
md = re.sub(r'[\x00-\x08\x0b\x0c\x0e-\x1f]', '', md)
assert '\x08' not in md
with open("ProposalResult.md", "w", encoding="utf-8") as f:
    f.write(md)

# 2. Convert markdown to a BODY-ONLY docx, template supplies the styles (NOT --toc)
pypandoc.convert_file(
    "ProposalResult.md", to="docx", outputfile="body.docx",
    format="gfm",                              # parse Markdown pipe tables
    extra_args=[f"--reference-doc={TEMPLATE}"],
)

# 3. Merge body INTO a copy of the template (keeps cover page + TOC field + headers/footers)
from docxcompose.composer import Composer
master = Document(TEMPLATE)                     # a fresh copy; never overwrite the shared template
# If the template ships trailing boilerplate that must stay AFTER the proposal body,
# split at a marker instead of appending; default templates are cover + TOC only.
composer = Composer(master)
composer.append(Document("body.docx"))
composer.save(OUT)

# 4. Disable template auto-numbering on heading styles (manual md numbers are authoritative)
doc = Document(OUT)
for style in doc.styles:
    ppr = style.element.find(qn('w:pPr'))
    if ppr is not None:
        for numpr in ppr.findall(qn('w:numPr')):
            ppr.remove(numpr)
first_h1_seen = False
for p in doc.paragraphs:
    name = (p.style.name or '') if p.style else ''
    if 'eading' in name or 'berschrift' in name:
        ppr = p._p.find(qn('w:pPr'))
        if ppr is not None:
            for numpr in ppr.findall(qn('w:numPr')):
                ppr.remove(numpr)
    # 5. Page break before each top-level chapter except the first (readability)
    if name in ('Heading 1', 'Überschrift 1'):
        if first_h1_seen:
            br = OxmlElement('w:br'); br.set(qn('w:type'), 'page')
            run = OxmlElement('w:r'); run.append(br)
            p._p.insert(0, run)
        first_h1_seen = True
doc.save(OUT)

# 6. then: fill cover-page addressee with client_name via python-docx (cover page is preserved)
```
